Generated by `bin/build_coverage_matrix`.

not against every RPC installed on a broker. Add one-RPC-per-line allowlists to
`data/pillar_allowlists/` and broker dumps to `data/broker_dumps/`, then run
`rake coverage:matrix` to refresh this document.
